\section{Netscape Communicator の使い方}

現在、 UNIX の世界で一番メジャーだと思われるブラウザが
 Netscape Communicator である。

 Netscape は、 UNIX の場合、完全に個人で環境設定でき、
ブックマークの使用も問題なくできる。

\subsection{Netscape で日本語入力}

たとえば、検索エンジンなどのページで、
ダイアログ・ボックスに日本語を入力し、
検索を実行したくなることがある。
Netscape では\footnote{正確には Netscape に限らず、
X Windows System のアプリケーション一般も同様}、
XIM プロトコルを通じて \texttt{kinput2} というソフトを
使って日本語入力を使用することができる。

\texttt{kinput2} による日本語入力の on / off は、
 \KEYTOP{Shift}+\KEYTOP{スペース} である。
日本語変換には Mule と同様に Canna が使われている。
 on / off が、この場合は \KEYTOP{Shift}+\KEYTOP{スペース}、
Mule の場合は「\texttt{c-o}」ということ以外はほとんど同じ操作である。

\begin{figure}[H]
\centering{\includegraphics[scale=0.7]{nskinput201.eps}}
\end{figure}

\section{テキスト・ブラウザ w3m}

w3m は伊藤彰則さんの作られたテキスト・ブラウザだ。
このブラウザは、最近のバージョンは画像も表示可能だが、
元々は基本的に文字だけで Web ページを表示するものだった。
文字の表示だけだと、非常に高速に Web ページを表示することが可能になり、
ストレスなく Web ブラウジングが可能になる。

起動するには、次のようにする。

\begin{progls}
% \slbfl{w3m -v}
\end{progls}

すると、次のような画面になって、 w3m が起動する。
使い方は、基本的に \texttt{jless} などのページャーと同様である。

\begin{progls}
                                \textcolor{blue}{Welcome to w3m!}

              \textcolor{blue}{This is w3m version w3m/0.2.3.2+cvs-1.196-img-1.17}
                            \textcolor{blue}{Written by Akinori Ito}



















\colorbox{black}{\textcolor{white}{≪↑↓Viewing <W3M startup page>}}
\end{progls}

Web ページを見るには次のようにする。
まず、「\texttt{U}」と打つ。
すると、画面一番下に URL を入力できるようになる。

\begin{progls}
Goto URL:
\end{progls}

ここにたとえば、 \texttt{http://www.aomori-u.ac.jp/} などの URL
を指定して \KEYTOP{Enter} する。

\begin{progls}
Goto URL: \slbfl{http://www.aomori-u.ac.jp/}
\end{progls}

すると、下の図のように、画像まで読み込んでページを表示してくれる。
