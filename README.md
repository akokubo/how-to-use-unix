# UNIXの使い方入門：ステップ・バイ・ステップガイド

この文書では、WindowsでLinux環境を提供する**WSL**（Windows Subsystem for Linux）の上で動く**Ubuntu**（ウブントゥ）と、Appleの**macOS**（マックオーエス）を例に、UNIX系のシステムの基本的な使い方を学ぶ。
コマンドライン操作に慣れていない人でも、一つ一つのステップを追いながらUNIXの基礎を理解できるように構成されている。

## 目次

### 第1部：UNIXの基本を学ぼう

*   **[UNIXシステムとは何か](unix-01.md)**: UNIXの歴史や特徴、基本的な考え方について学ぶ。
*   **[UNIXシステムを使ってみよう](unix-02.md)**: ターミナルの起動方法や、コマンド実行など、UNIXシステムに触れる第一歩。
*   **[ファイルとディレクトリ操作の基本](unix-03.md)**: `ls`、`cd`、`pwd` など、ファイルとディレクトリに関する基本的なUNIXコマンドの使い方を学ぶ。

### 第2部：ファイルとディレクトリの操作

*   **[ファイル操作](unix-04.md)**: ファイルの作成、コピー、移動（名前の変更）、削除など、ファイル操作に必要なコマンドを学ぶ。
*   **[ディレクトリとパス](unix-06.md)**: ディレクトリ（フォルダ）の概念と操作方法、ファイルの場所を表すパスについて理解を深める。
*   **[アクセス権限（パーミッション）](unix-08.md)**: ファイルやディレクトリのアクセス権（パーミッション）について学び、セキュリティの基礎を理解する。

### 第3部：UNIXを使いこなす

*   **[ページャー](unix-05.md)**: 長い出力結果を1画面ずつ表示するページャーの使い方を学ぶ。
*   **[プロセスとジョブ](unix-07.md)**: 実行しているプログラム（プロセス）の確認方法や、ジョブの管理について学ぶ。
*   **[シェル入門](unix-09.md)**: コマンド入力を簡単にするシェルについて理解を深め、より高度な操作を目指す。
*   **[ネットワーク機能](unix-10.md)**: ネットワーク関連のコマンドや機能について触れる。
